# Return

## **Return Policies and Controls**



To maintain accuracy and prevent misuse, several controls are applied when processing returns:

- **Return time window:** Returns can only be processed within the allowed time period defined in the **Return Settings**. If a sale is older than the permitted window, the return will not be allowed.
- **Over-return prevention:** The system prevents returning more quantity than what was originally purchased.
- **Partial returns:** You can return only selected items or quantities from a sale without returning the entire transaction.
- **Products without stock tracking:** Items that do not track inventory can still be returned, but stock levels will not be affected.
