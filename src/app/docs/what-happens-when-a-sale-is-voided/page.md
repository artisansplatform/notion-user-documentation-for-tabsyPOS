# Sales Transactions

## What Happens When a Sale is Voided


- Sale status changes to "Voided" with red badge
- Total amount is shown with strikethrough
- Stock is automatically restored for all items in the sale
- Inventory update logs are created for each restored item
- Voided sales are excluded from sales statistics (today/month/year totals)
- Receipt can still be printed - shows "VOIDED" watermark
