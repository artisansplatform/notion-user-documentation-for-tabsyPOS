# Settings

## Return Settings


- **Return Window**: How many days after a sale a return is allowed
    - Presets: Unlimited (no limit), No Returns (disabled), 7 days, 14 days, 30 days
    - Custom number of days
- **Default Restock**: Whether returned items are automatically added back to inventory (enable/disable checkbox)
- **Require Return Reason**: Whether a reason must be provided when creating a return (enable/disable checkbox)
