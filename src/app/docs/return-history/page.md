# Sales Transactions

## Return History



The **Return History** page provides a complete record of all product returns processed in your store. It helps you track refund activity, review return details, and maintain accurate records for customer service and reporting.


To access this section, go to the **Return History** menu. The page displays a list of all return transactions along with key details such as the **return number**, **original bill number**, **date**, **customer**, **reason for return**, and the **refund amount**.


At the top of the page, you will find quick filters that help you easily navigate through return records:

- **All:** Displays all return transactions.
- **Active:** Shows valid returns that are currently recorded in the system.
- **Voided:** Displays returns that have been cancelled or voided.

You will also see two summary cards that provide a quick overview of return activity:

- **Today’s Returns:** Total value of returns processed today.
- **This Month:** Total value of returns processed during the current month.

These statistics give you a quick snapshot of refund activity without affecting the detailed return list below.


Each entry includes a **status badge** indicating whether the return is active or voided.


You can click on any return to **view the complete return details**, including the items returned and the refund information associated with that transaction.
