# Settings

## AI Model




TabsyPOS offers **OpenRouter** AI model options that power **intelligent product search** and **AI-powered cart creation**.

> **Note:** Voice recognition is browser-based and does not rely on any AI model.

### **OpenRouter Model**


This option allows you to use AI through **OpenRouter’s platform** by configuring your own **OpenRouter API Key.**


TabsyPOS uses **free OpenRouter models**, meaning you do **not need to pay anything** to use this feature.


>   
> **Important:** Because OpenRouter is an online third-party integration, AI features will not work when **you are offline**.  
> 

Follow the steps below to set up the OpenRouter API Key:

1. Go to [https://openrouter.ai/](https://openrouter.ai/)
2. Log in to your account.
3. Open your **Profile**.
4. Navigate to the **Keys** section.

![Screenshot_2025-12-08_111345.png](/images/docs/settings/ai-model/2025-12-08T05:45:00.000Z.png)

- Click **Create API Key**.

![OpenRouter-12-08-2025_11_15_AM.png](/images/docs/settings/ai-model/2025-12-08T05:46:00.000Z.png)

1. Enter a **Name** for your key.
2. Set **Credit Limit** (Recommended: **0**).
3. Leave **Reset Limit Every** as the default.
4. Leave Expiration as the default.
5. Click on Create

![ee03bb2e-7496-4a40-b8d6-fa79e11b3970.png](/images/docs/settings/ai-model/2025-12-08T05:49:00.000Z.png)

- Copy the generated key (Note: It will be shown **only once**).

![OpenRouter-12-08-2025_11_19_AM_%281%29.png](/images/docs/settings/ai-model/2025-12-08T05:50:00.000Z.png)

- In **TabsyPOS**, go to **Settings → AI Model**.

![TabsyPOS-Smart-POS-System-12-08-2025_11_04_AM.png](/images/docs/settings/ai-model/2025-12-08T05:35:00.000Z.png)

- Click **API Key** and paste your key and click Save.

    ![TabsyPOS-Smart-POS-System-12-08-2025_11_03_AM.png](/images/docs/settings/ai-model/2025-12-08T05:33:00.000Z.png)


Your AI model setup is now complete and ready to use.
