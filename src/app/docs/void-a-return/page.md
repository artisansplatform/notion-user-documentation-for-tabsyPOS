# Return

## Void a Return



If a return was created by mistake, it can be **voided**.


Voiding a return reverses the effects of that return transaction and ensures your records remain accurate.

- Returns can be voided from the Return Details modal
- Voiding reverses stock changes (if items were restocked)
- Voided returns show red "Voided" badge and struck-through styling
- Voided returns do NOT count toward returned quantities (allows re-returning)
